# Slot 777 - স্লট মেশিন গেম

একটি মোবাইল-ফ্রেন্ডলি স্লট মেশিন গেম HTML, CSS, এবং jQuery দিয়ে তৈরি।

## ফাইল স্ট্রাকচার

```
slot777/
├── index.html      # মূল HTML ফাইল
├── style.css       # CSS স্টাইলশীট
├── script.js       # jQuery স্ক্রিপ্ট
├── images/         # ইমেজ ফোল্ডার
│   ├── seven.png   # সেভেন সিম্বল
│   ├── diamond.png # ডায়মন্ড সিম্বল
│   ├── bell.png    # বেল সিম্বল
│   ├── bar.png     # বার সিম্বল
│   ├── star.png    # স্টার সিম্বল
│   ├── cherry.png  # চেরি সিম্বল
│   └── coin.png    # কয়েন ইমেজ
└── README.md       # এই ফাইল
```

## কিভাবে ব্যবহার করবেন

1. পুরো `slot777` ফোল্ডারটি ডাউনলোড করুন
2. `index.html` ফাইলটি ব্রাউজারে ওপেন করুন
3. অথবা যেকোনো ওয়েব সার্ভারে আপলোড করুন

## কাস্টমাইজেশন

### জেতার সম্ভাবনা পরিবর্তন
`script.js` ফাইলে `WIN_CHANCE` ভেরিয়েবল পরিবর্তন করুন:
```javascript
const WIN_CHANCE = 0.10; // 10% জেতার সম্ভাবনা
```

### রঙ পরিবর্তন
`style.css` ফাইলের শুরুতে CSS variables পরিবর্তন করুন।

## প্রয়োজনীয়তা

- আধুনিক ওয়েব ব্রাউজার (Chrome, Firefox, Safari, Edge)
- jQuery 3.x (CDN থেকে লোড হয়)
- ইন্টারনেট কানেকশন (Google Fonts এবং jQuery এর জন্য)
